# tasklistapp

## Инструкция по разворачиванию:

### Сервер
- перейти в ./server и вызвать
`npm install`
- при возникновении ошибок вызвать 
`npm install --legacy-peer-deps`
- вызвать
`node server.mjs`
или использовать скрипт в package.json 
`start server`


### Клиент
- перейти в ./client и вызвать
`npm install`
- при возникновении ошибок вызвать
`npm install --legacy-peer-deps`
- вызвать
`vue-cli-service serve`
или использовать скрипт в package.json 
`start client`
